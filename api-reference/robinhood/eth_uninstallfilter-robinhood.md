# eth\_uninstallFilter - Robinhood

Uninstalls a filter by ID. Filters also expire automatically if not polled for some time — explicit cleanup is good practice.

## Parameters

| Parameter | Type     | Required | Description                                            |
| --------- | -------- | -------- | ------------------------------------------------------ |
| filterId  | QUANTITY | Yes      | Filter ID returned by a previous `eth_new*Filter` call |

## Response Parameters

| Field  | Type    | Description                                                      |
| ------ | ------- | ---------------------------------------------------------------- |
| result | boolean | `true` if the filter was uninstalled, `false` if it didn't exist |

## Use Cases

* Resource cleanup when a polling loop ends
* Preventing filter accumulation in long-running services

## Error Handling

| Status Code | Error Message     | Cause                                          |
| ----------- | ----------------- | ---------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`            |
| -32602      | Invalid params    | Request parameters are missing or malformed    |
| -32603      | Internal error    | Server-side error while processing the request |
| 429         | Too Many Requests | Rate limit exceeded for your plan              |
